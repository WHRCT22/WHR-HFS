关于WHR-HFS

本程序由 @王浩瑞 开发，您当前运行的版本为2024年1月24日编译

本程序的功能是在局域网或广域网传输文件以及聊天

本程序基于NodeJS驱动，使用了以下的框架:

  • Express（Express 是一个Node.js Web 应用程序开发框架,快速方便地创建强大的 API。）
  • Multer（Multer是一个 Node.js 中间件,用于处理 multipart/form-data 类型的表单数据,它主要用于上传文件。）
  • WebSocket（Websocket是一种用于H5浏览器的实时通讯协议,可以做到数据的实时推送。）
  • Qrcode-terminal（一个在控制台生成二维码的模块。）
  • Http-Proxy-Middleware（Http-Proxy-Middleware用于后台将请求转发给其他域。）
  • NodeJS中的自带函数及框架
